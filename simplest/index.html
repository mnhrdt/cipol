What is this?

This is a tool to evaluate image processing algorithms online. You write your program in C and we run it in our computer. If the compulation and execution are successful, you will see the output of your program and any PNG images that it creates will be displayed.

The program is compiled and run in a controlled environment, where it can probably do no harm, even if it tries to. The program is allowed to run for at most 10 seconds of CPU time and it can use at most 100MB of memory.
